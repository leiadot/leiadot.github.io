二、指令

v-if、v-show

v-if：可做條件判斷，若為 false 會將整個DOM結構拿掉。
v-else：搭配 v-if ，條件滿足後將顯示資料。範例
v-show：可作條件判斷，若為 false 資料不顯示，但會保存 Dom 結構。

v-model

v-model：可作雙向資料綁定，更改 input 的值， DOM 也會即時更改資料。
