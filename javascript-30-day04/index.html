stringObj.split([separator[, limit]])

  • 回傳為字串分割成的字串陣列。
  • stringObj 必要項。要分割的 String 物件或字串常值。 split 方法不會修改這個物件。
  • separator 選擇項。字串或規則運算式物件，可識別用來分隔字串的一個或多個字元。如果省略，則會傳回包含整個字串的單一元素陣列。
  • limit 選擇項。用來限制陣列中所傳回元素個數的值。
